***What is web development ?***

In simple words, web development is creating a single page or pages. For instance, Twitter which has many HTML pages. Whilst a single-page website has one HTML page with endless scrolling of contents.


***How do websites work ?***

Let’s have a look at a brief description of how a website work. As your Home Screen of your computer has files just the same is true for websites, which also host files on a server – an additional computer. Noting that many servers are linked together over the internet is also important. And for this reason, when you click on a link, it loads the information on your computer from that server. And in this case, your computer serves as the web server because you are using it to visit the website. While your browser serves as the client.

***HTML*** is an acronym that stands for Hyper-text Mark-up Language. It is made up of tags like headings, paragraph etc that when combine produce a bare-born document.

```
<h1> H1 stands for Heading 1 </h>
<p> P stands for Paragraph 1 </p>
<i> I stands for Itallic </i>

```

> The above codes will produce this output.

```
H1 stands for Heading 1
P stands for Paragraph 1
I stands for Itallic
```

> Tags help us format our website document to look how we want them.
