# ALU GDSC

Welcome to the official GitHub repository of the ALU GDSC blog website! 

Our Google Developer Student Club is dedicated to empowering students with the knowledge and skills they need to succeed in the digital world. We offer a wide range of resources, including weekly how-to articles that cover everything from web development to app creation and artificial intelligence.

### Visit our website

To visit our website, please click [here](https://santigio.github.io).

### Contribute

We welcome contributions from developers of all skill levels. If you would like to contribute to our blog website, please follow these steps:

1. Fork this repository
2. Create a new branch for your changes (e.g. `feature-article`)
3. Make the necessary changes
4. Submit a pull request to the `master` branch

Please ensure that your changes are well-documented and follow our code of conduct.

### Code of Conduct

All contributors are expected to adhere to our code of conduct. 

### Styling

Please use the following markdown styling when making contributions:

- Headers: `#`
- Sub-headers: `##`
- Lists: `-`
- Links: `[Link Text](Link URL)`
